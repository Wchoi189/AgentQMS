When creating new artifacts, follow these rules:

1. **File Naming**: Use format YYYY-MM-DD_HHMM_[TYPE]_descriptive-name.md
2. **Directory Placement**:
   - Templates → templates/
   - Implementation plans → implementation_plans/
   - Assessments → assessments/
   - Research → research/
   - Design docs → design_documents/
   - Bug reports → bug_reports/
3. **Frontmatter**: Always include structured frontmatter with title, date, type, category, status, version, and tags
4. **Naming Conventions**: Use kebab-case for descriptive names, proper prefixes for file types
